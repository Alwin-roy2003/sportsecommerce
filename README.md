# Sports Ecommerce Website

A Django-based ecommerce platform for sports products.

## Features
- User authentication
- Product categories
- Cart & wishlist
- Order management
- Admin dashboard

## Tech Stack
- Django
- HTML, CSS, JavaScript
- SQLite

## Setup
```bash
git clone https://github.com/Alwin-roy2003/sportsecommerce.git
cd sportsecommerce
python -m venv venv
pip install -r requirements.txt
python manage.py runserver
